# Demo: Campus Lab Safety Risk Inspection

This is a privacy-safe simulated run that shows the intended handoff shape for `ppt-image-share-builder`.

The topic, sources, slide images, and script are synthetic. They do not contain private classroom materials, real student names, campus names, logos, or unpublished files.

In a real workflow, the slide images should be generated with image2 from the per-slide prompts. The placeholder images here exist so the public repository can demonstrate the full chain:

```text
input notes -> image2 outline -> slide images -> contact sheet -> PPTX assembly -> talk script
```

## Files

- `input-notes.md`: synthetic source notes
- `image2-outline.md`: sample per-slide image2 prompt plan
- `10-minute-script.md`: sample timed talk script
- `images/`: placeholder slide images standing in for image2 outputs
- `contact-sheet-demo.jpg`: overview of the generated slides
