# Demo: Lab Safety Inspection

This is a privacy-safe demo that shows the handoff shape for `ppt-image-share-builder`.

The topic, sources, slide images, and script are synthetic. They do not contain private classroom materials, real student names, or unpublished files.

In a real workflow, the slide images should be generated with image2 from the per-slide prompts. The placeholder images here exist only so the public repository can demonstrate contact-sheet QA, GIF preview, and PPTX assembly without exposing private course work.

## Files

- `input-notes.md`: fake source notes
- `image2-outline.md`: sample per-slide outline and prompt plan
- `10-minute-script.md`: sample timed talk script
- `images/`: placeholder slide images standing in for image2 outputs
- `contact-sheet-demo.jpg`: overview of the generated slides
